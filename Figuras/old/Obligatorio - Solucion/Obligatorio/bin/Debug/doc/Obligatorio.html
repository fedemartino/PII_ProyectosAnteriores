Classes

Class Description
Coordenadas Clase que implementa la interfaz IPoint. Se encarga de retornar las coordenadas de un punto en el plano.
Cuadrado Clase que implementa la interfase de tipo polígono. Tiene la responsabilidad de calcular el área y el perímetro de determinado cuadrado.
Dibujo Clase que implementa los objetos del tipo Dibujo. Cada dibujo tendrá un identificador y una lista de componentes (figuras)
Downloader Descarga archivos de una ubicación de la forma "http://server/directory/file" o "file:///drive:/directory/file"
ExtraerTags Esta clase tiene la responsabilidad de buscar los tags que se encuentran dentro de un texto. Debe conocer los caracteres con los que se delimitan los tags.
Parser Esta Clase tiene la responsabilidad de buscar el nombre y los atributos, con su valor correspondiente, dentro de determinado tag. Debe conocer los caracteres con los que se delimitan las distintas partes que conforman un tag.
Pausar Clase que implementa la interfaz IAction. Se encarga de pausar las acciones el tiempo indicado
Pintar Clase que implementa la interfaz IAction. Se encarga de pintar los dibujos en determinado color en el board
Poligono Clase abstracta que implementa las interfaces IShape e Itransformable. Esta clase define el tipo de todos los poligonos.
ProcesadorTags Clase encargada de analizar un poligono.
Rectangulo Clase que implementa la interfase de tipo polígono. Tiene la responsabilidad de calcular el área y el perímetro de determinado rectángulo.
Rotacion Clase encargada de realizar la rotacion.
SimetriaEjeX Clase encargada de realizar la simetria de eje X.
SimetriaEjeY Clase encargada de realizar la simetria de eje Y.
Tags Esta clase tiene las responsablidad de conocer los datos de un objeto.
Transformar Clase que implementa la interfaz IAction. Se encarga de transformar los dibujos a partir de alguna transformación
Traslacion Clase encargada de realizar la traslación.
Triangulo Clase que implementa la interfaz de tipo polígono. Tiene la responsabilidad de calcular el área y el perímetro de determinado rectángulo.
